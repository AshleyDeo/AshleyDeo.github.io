Game Programming Github Repository

All work done using C++ with OpenGL and SDL

Pong

I recreated the game Pong. It is 2 player and has a scoreboard.

Space Invaders

Clone of Space Invaders using textures. It has a score tracking how many rounds a player has won. The original one did not have sound. Homework 6 was to choose one of the previous projects and add sound to it.

Platformer Demo

This was used to learn how to create and import tilemaps, create player physics and animation of sprites. The tilemapping was hard at first but very useful for the final project.

MegaMan Starforce Clone

Using everything we learned in class, we had to create our own game. I recreated Mega Man Starforce and Mega Man Battle Network. I took sprites from the game and created the Battle Network battle system and an overworld like in Starforce. There are 10 enemies in the overworld and each one has 3 randomized enemies to fight. I took soundtracks for the games and edited them to loop.
